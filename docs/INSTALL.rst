wcc.committee Installation
--------------------------

To install wcc.committee using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.committee`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.committee

* Re-run buildout, e.g. with:

    $ ./bin/buildout
